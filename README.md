# Adult Census Income Prediction

Binary classification project predicting whether an individual's annual income
exceeds $50,000, using the [UCI Adult (Census Income) Dataset](https://archive.ics.uci.edu/dataset/2/adult).

## Research Questions

- Which demographic and socioeconomic factors are most strongly associated with
  higher income?
- How well can we predict income class from census features?
- What actionable insights emerge for education and public policy?

## Dataset

- **Source:** 1994 U.S. Census (Barry Becker extraction)
- **Records:** 48,842 individuals
- **Features:** 14 (categorical + integer)
- **Target:** `income` — binary (`<=50K` / `>50K`)

The raw dataset is included at `data/raw/adult_income.csv`.

## Usage

### Full pipeline (command line)

```bash
python main.py
```

This runs three steps in order:
1. **Clean data** — handles missing values, whitespace, duplicates; saves
   `adult_income_clean.csv` and `adult_income_featured.csv`
2. **Train logistic regression** — baseline model with balanced class weights
3. **Train random forest** — ensemble model (200 trees, max depth 15)

Outputs are saved to `models/` (`.joblib` pipelines) and `reports/`
(JSON metrics + confusion matrix PNGs).

### Notebooks (interactive analysis)

Run `python main.py` first to generate the processed data, then open the
notebooks in order:

| Notebook | Purpose |
|---|---|
| `data_check.ipynb` | Quick sanity check of clean and featured datasets |
| `explore.ipynb` | Exploratory data analysis, chi-square tests, ANOVAs |
| `modeling.ipynb` | Model training, evaluation, coefficient analysis, comparison |

## Feature Engineering

Several raw categorical variables are collapsed into broader groups to reduce
sparsity and improve interpretability:

| Original Column | Grouped Column | Example Groups |
|---|---|---|
| `education` | `education_grouped` | Before HS, High School, Some College, Bachelors, Masters, Doctorate, Professional |
| `marital-status` | `marital_grouped` | Married, Never Married, Previously Married |
| `native-country` | `region` | North America, Latin America & Caribbean, Europe, Asia, Unknown |
| `age` | `age_group` | Young (≤20), Early Career, Mid Career, Senior, 60+ |
| `hours-per-week` | `hours_group` | ≤10, 11–25, 26–40, 41–60, 60+ |

## Key Findings

- **Education** is the strongest and most actionable predictor of income.
  Lower educational attainment (Before HS, High School) strongly predicts
  ≤50K, while Doctorate and Professional degrees strongly predict >50K.
- **Marital status** shows a strong association but the causal mechanism is
  unclear (likely confounded by selection effects).
- **Occupation type** matters — executive/managerial roles favor >50K while
  farming/fishing and service roles favor ≤50K.
- Both models achieve ~82% accuracy. Random forest slightly outperforms
  logistic regression on F1-score.
